Panorama Stitching — Variant

Objective
- Explore an alternative configuration of the panorama pipeline to compare behavior against the main approach.

What’s different
- This variant experiments with alternative parameter choices (e.g., detector settings, ANMS target count, descriptor/matching thresholds) and minor implementation differences in the matching/visualization flow.

Data
- Place image sequences under: projects/panorama-stitching-variant/data/
- Environment overrides (optional): same as the main panorama project if used in the notebook (PANORAMA_DATA_DIR, PANORAMA_RESULTS_DIR).

How to run
- Open notebooks/panorama-stitching-variant.ipynb in Jupyter.
- Follow the same general steps as the main panorama project; see that README for context.

Notes
- This folder exists to highlight algorithmic experimentation; it is not a duplicate for grading.
